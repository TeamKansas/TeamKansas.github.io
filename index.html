About Me...

Hi there! My name is Micah.
I am currently in the 11th grade, and I enjoy computer programming, rock-climbing, hiking, and mathematics. I spend most of my free time rock-climbing or writing some computer program.

I participate in climbing competitions sanctioned by USA Climbing, but I live in Kansas so I do not climb on real rocks very often.

My Programming career in a nutshell

  • 5th grade: starting programming with Python
  • 7th grade: made a small Mandelbrot Set viewer with Python
  • 8th grade: Switched to C, made a bigger, faster Mandelbrot Set viewer using GLUT framework
  • 9th grade: made a maze generator, Tetris, Snake, Conway's Game of Life, and Minesweeper with GLUT
  • 10th grade: GLUT no longer supported, made tiffDrawingTools to draw to uncompressed TIFF files
  • 10th grade: made random terrain map generator, maze generator, and 2d L-System generator
  • 10th grade: made program that finds "interesting" spots on the Mandelbrot Set
  • 10th grade: made command-line Tetris, Snake, Minesweeper, and Wordle, prime number program
  • 10th grade: learned javascript and html, made even better interactive Mandelbrot set viewer, and Conway's game of life
  • Summer before 11th grade: made an even better 2d L-System generator with javascript and html
  • 11th grade: started learning about neural networks

I learned Python and C from some books, and I learned html and javascript from the MDN Web docs

Note: I left out many programming projects that I have worked on over the years, as they were less interesting or less successful. It is important to note that these failures were crucial to helping me become a better programmer. I still have a lot to learn when it comes to algorithms and program flow, but I'm getting there.
